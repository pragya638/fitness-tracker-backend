# Fitness Tracker Backend API

A production-ready backend application built using Spring Boot, Spring Security, JWT Authentication, PostgreSQL, Docker, and Render Deployment.

## Live API

Backend URL:
https://fitness-track-backend-vcwk.onrender.com

Swagger Documentation:
https://fitness-track-backend-vcwk.onrender.com/swagger-ui/index.html

---

## Project Overview

The Fitness Tracker Backend API provides secure REST APIs for user authentication, activity management, recommendations, and future fitness tracking features.

The application follows modern backend development practices including JWT-based authentication, layered architecture, API documentation, and cloud deployment.

---

# Features

* JWT Authentication
* Spring Security
* RESTful APIs
* PostgreSQL Database Integration
* Swagger/OpenAPI Documentation
* Docker Support
* Cloud Deployment on Render
* Role-Based Access Architecture
