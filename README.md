## API Endpoints

### Authentication APIs

#### Register User

```http
POST /api/auth/register
```

#### Login User

```http
POST /api/auth/login
```

---

### Activity APIs

#### Create Activity

```http
POST /api/activities
```

#### Get All Activities

```http
GET /api/activities
```

#### Update Activity

```http
PUT /api/activities/{id}
```

#### Delete Activity

```http
DELETE /api/activities/{id}
```

---

### Recommendation APIs

#### Generate Recommendation

```http
POST /api/recommendation/generate
```

#### Get Recommendations By User

```http
GET /api/recommendation/user/{userId}
```
